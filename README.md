# x-lisp

## My Motive

(1) Just learn how to write compiler
for a language like scheme,
it is a core skill.

(2) x-lisp to geometer and canvas,
as the elisp to emacs.

(3) Use my-lisp to read EOPL,
so that even the simple part will not be boring.
